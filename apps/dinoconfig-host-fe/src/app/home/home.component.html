How it works

Get started with DinoConfig in three simple steps

1

Design

Use our visual editor to create your configuration schema with custom types and validation rules.

type: "object"
properties: {
feature_flags: {...}
api_settings: {...}
}
2

Fetch Configurations

Use our SDK to retrieve your app's configuration in JSON format.

async ngOnInit() {{ '{' }}
  const sdk = new DinoConfigSDK();
  await sdk.configure({{ '{' }}
    apiKey: 'your-api-key'
  {{ '}' }});
  const config = await sdk.getConfigAPI()
    .getConfigValue('BrandName', 'ConfigName', 'isConfigEnabled');
{{ '}' }}
3

Use your configurations

Check a feature flag in your frontend or backend and enable features conditionally.

if (config.BrandName.ConfigName
    .isConfigEnabled) {
  // Enable the feature
  enableFeature();
} else {
  // Fallback behavior
  disableFeature();
}
